# if 语法

if 条件:
    条件成立执行的代码1
    条件成立执行的代码2

# if else 语法
if 条件:
    条件成立执行的代码1
    条件成立执行的代码2
    ......
else:
    条件不成立执行的代码1
    条件不成立执行的代码2
    ......

# 总结

- if语句语法

``` python
if 条件:
    条件成立执行的代码
```

- if...else...

``` python
if 条件:
    条件成立执行的代码
else:
    条件不成立执行的代码
```

- 多重判断

``` python
if 条件1:
    条件1成立执行的代码
elif 条件2:
    条件2成立执行的代码
else:
    以上条件都不成立执行的代码
```

- if嵌套

``` python
if 条件1:
    条件1成立执行的代码
    if 条件2:
        条件2成立执行的代码
        ....
```
